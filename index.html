About us

Hello, we are two upcoming bachelors who study Medialogy at Aalborg University in Denmark. At the moment we are in our 6th semester, and this portfolio will showcase our combined work from the previous five semesters, of which we have chosen six of our most interesting projects. These can be navigated to by hovering over "Projects" tab in the navigation bar at the top of the screen, or simply by scrolling downward.

This project is from 1st semester Medialogy 2015, where we created an android app, which should act as a digital timecapsule.

This was a semester project from 4th semester Medialogy in 2017, where a speech-based home automation system was designed and implemented using an Open Source Voice Assistant known as Mycroft AI, and tested using the system usability scale to determine user satisfaction.

This 3rd semester project from 2016 revolved around image processing, were we created an augmented boardgame, using a porjector and a webcam.

Bastille is a semester project from 5th semester Medialogy in 2017, which regarded whether the aspect of interaction increased immersion in VR experiences.

This 4th semester project was an attempt on using audio processing to create an automated baby carriage, that would start swaying when the baby tarted to cry.

DayChaser was a miniproject in the Theory and Practice of Game Design and Development Medialogy course in spring 2018. The game revolved around picking up lights to turn night into day, and then battling a monster that was previously hidden in the dark. A playable prototype of the game was finished and showcased, and a trailer was created for it.
