About us

Hello, we are two upcoming bachelors who study Medialogy at Aalborg University in Denmark. At the moment we are in our 6th semester, and this portfolio will showcase our combined work from the previous five semesters. Below we feature a showcase of products from either courses or semester projects.

This project is from 1st semester Medialogy 2015, where we created an android app, which should act as a digital timecapsule.

This was a semester project from 4th semester Medialogy in 2017, where a speech-based home automation system was designed, implemented and tested using the system usability scale.

This 3rd semester project from 2016 revolved around image processing, were we created an augmented boardgame, using a porjector and a webcam.

Bastille is a 5th semester Medialogy project regarding whether the aspect of interaction increased immersion in VR experiences.

This 4th semester project was an attempt on using audio processing to create an automated baby carriage, that would start swaying when the baby tarted to cry.
